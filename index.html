下载与安装

从下载理工学堂助手的最新 EXE 版本，点开即用。

登录账号

打开应用程序，输入你的理工学堂账号和密码进行登录。可以选择记住密码，方便下次快速登录。

浏览课程

登录成功后将展示你所选课程的列表。点击任意课程即可查看该课程已发布的作业。

查看作业

在课程页面中，你可以查看该课程的所有作业。点击"查看题目"即可浏览作业详情。

导出题目

你可以将题目导出为 Word 或 PDF 格式，方便离线查看和打印，记得在设置中选择导出选项。

设置与个性化

在设置页面，你可以修改导出路径、选择导出格式，以及查看你的账户信息。

环境准备

确保你的电脑已安装 Python 3.7 或更高版本。建议从 Python 官网下载并安装。

下载源码

下载理工学堂助手的 Python 文件。

安装依赖

在项目中下载 requirements.txt 文件，运行 pip install -r requirements.txt 安装所需的依赖包。

运行程序

在命令行中运行 python 理工学堂助手.py 启动程序。然后输入你的理工学堂账号和密码。

使用功能

按照程序提示，选择相应的功能，如浏览课程、查看作业、导出题目等。

自定义开发

Python 版本允许你根据需求修改源码，添加新功能或优化现有功能。
